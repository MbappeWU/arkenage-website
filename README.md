# MacSweep Website

MacSweep 官方分发网站 — [arkenage.com](https://arkenage.com)

Safety-first macOS storage cleaner. Reclaim disk space with confidence.

## Structure

```
index.html          Landing page (ZH/EN bilingual)
css/style.css       Dark glassmorphism theme
js/main.js          Scroll animations, language toggle, nav
img/                Screenshots & favicon
CNAME               Custom domain → arkenage.com
```

## Local Preview

```bash
python3 -m http.server 8000
# open http://localhost:8000
```

## Deploy

Hosted on GitHub Pages. Pushes to `main` branch auto-deploy via GitHub Actions.
